Trabajos.

.Desafío 1 Mi Curriculum Vitae en HTML {

Bienvenido a tu primer desafío, un desafío es una actividad integral que busca poner en práctica todo lo aprendido en la unidad y en el curso hasta el momento. A continuación, te invitamos a realizar una pequeña actividad que consiste en armar tu CV en HTML. Recuerda utilizar la estructura básica de HTML y todas las etiquetas que hemos aprendido en esta sesión. Siéntete libre de agregar el contenido que estimes conveniente.

}

.Desafío 2 - Construyendo un landing page {

En este desafío validaremos nuestros conocimientos de CSS trabajados en clase y en la guía. Para lograrlo, necesitarás implementar CSS utilizando archivos hojas de estilo externas al documento HTML y de base para homogeneizar el diseño del sitio. Además, deberás estructurar una página en términos y assets del proyecto, además de implementar bordes, margen, padding y selectores de etiqueta o clase según sea necesario.

}

.Desafío 3 - Iguana page {

En este desafío validaremos nuestros conocimientos de posicionamiento en CSS. Para lograrlo, necesitarás diferenciar los valores asociados a display (inline, block, inline-block), utilizar floats para posicionar texto e imágenes, y resolver el problema de colapso de floats utilizando un clearfix.

}
